## API简介

RightBTC为用户提供了稳定而实用的API接口，可实现获取行情、下单交易等功能。使用API接口可让程序代替人执行策略下单、套利和对冲交易。目前已有上百种策略在RightBTC安全运行。

### 请求步骤

- 根URL

  www.rightbtc.com

- 编码字符集

  UTF-8

- GET方法的header类型

  application/json

- POST方法的header类型

  application/json
